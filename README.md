# sticky-notes-application

## Description
The Sticky Notes App is a simple and intuitive web application that allows users to create, edit, and delete virtual sticky notes. It's designed to help users keep track of their tasks, ideas, and reminders in an organized and visually appealing way.

## Features
- Create new sticky notes with custom content.
- Edit existing notes to update tasks or reminders.
- Delete notes that are no longer needed.
- User-friendly interface with responsive design for seamless use across devices.

## Technology Stack
- **Frontend**: HTML, CSS (with Bootstrap), JavaScript for dynamic interactions.
- **Backend**: Django for server-side logic and handling CRUD operations.
- **Database**: SQLite for managing stored data.
